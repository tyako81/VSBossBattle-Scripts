# VSBossBattle-Scripts

ポートフォリオに掲載している3Dアクションゲーム「VS BossBattle」のC#スクリプトです。
※プロジェクトデータ全体ではなく、自身で実装したスクリプトファイルのみを公開しています。

## 開発環境
- Unity
- C#


### 1. ステートマシンによるボスの行動制御
**該当ファイル：`AttackField/FieldManager`**
ボスの行動パターン（待機、攻撃、移動など）をステートマシンで管理し、拡張しやすくバグの起きにくい設計にしました。また、プレイするたびにランダムに行動パターンが変化するロジックを組んでいます。

### 2. 追加簡単な敵の攻撃
**該当ファイル：`AttackField/FieldManager`**
敵の攻撃手段をリストで管理しているので、AttackedFieldの子オブジェクトに敵の攻撃手段を追加することによって簡単に敵が追加した攻撃を行ってくれるようになる。

### 3. キャラクター選択
**該当ファイル：`Player/Character/CharacterDatabase.cs、Player/Player.cs` など**
Classを用いキャラクターを選択できるようにしました。PlayerFabsを用いて簡易的なセーブを行いました。キャラクターはそれぞれ足の速さやHP、攻撃力が違うようになっています。キャラクターを選択できるようにすることで遊びの幅が増加しました。

### 4. 装備システム
**該当ファイル：`Player/Equipment/Equipment.cs` など**
キャラクターシステムのようにClassを用いて装備システムを作成しました。それぞれの装備がプレイヤーのステータスに影響し、ゲームの攻略手段が増加した。
